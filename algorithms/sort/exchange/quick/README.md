- <i id="2"></i>**`算法来源`**
`快速排序`是对[冒泡排序](../bubble)的一种改进。  
基本思想:通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的索引数据都比另外一部分的所有数据都要小,然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序序列。
